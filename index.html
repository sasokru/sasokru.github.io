Featured · drip.py

An infusion drip monitor

Arduino · C++ · MATLAB · Digital Health

drip.py measures the flow of a gravity infusion without touching the fluid. An infrared beam sits across the drip chamber; every falling drop breaks the beam, and the Arduino counts these interruptions to calculate the drop rate and convert it into millilitres per minute.

The live rate is shown on an OLED display, and an RGB LED gives an at-a-glance status. The idea comes straight from bedside nursing, where drop rates are still counted by eye and a wristwatch.

LOW < 0.67 mL/min OK 0.67–2.0 FAST > 2.0

Hardware

  • Microcontroller Arduino Uno
  • Drop detection IR break-beam sensor
  • Display SSD1306 OLED (I²C)
  • Status signal RGB LED

Software

  • Firmware C++ (Arduino IDE)
  • Display library U8g2
  • On-device smoothing Exponential moving average
  • State estimation Kalman filter (MATLAB)
